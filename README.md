# first-repo

This is a simple HTML profile page.

## How to run

1. Open the file `1st.html` in VS Code.
2. Right-click and select "Open with Live Server" to preview in browser.

## Extensions installed

- Live Server: For live preview
- Prettier: For code formatting
- Auto Rename Tag: For HTML tag renaming
- HTML CSS Support: Already installed

## Git Bash Setup (Added by BLACKBOXAI)

To set up Git Bash on Windows 10:

1. Download from https://git-scm.com/download/win (64-bit).
2. Run installer as Admin.
3. Key options: PATH "Git from cmd & 3rd-party", Terminal "MinTTY".
4. Open via Start > Git Bash or right-click "Git Bash Here".
5. Verify: `git --version`.
6. VSCode: Set as default terminal (Ctrl+Shift+P > Terminal: Select Default Profile).
